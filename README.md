# TikTalk

TikTalk - это мой первый проект на Angular. Здесь я учусь и реализую новые, и современные подходы к frontend разработке. Само приложение является социальной сетью для разработчиков, где они могут общаться в чате или создавать посты на различные темы для обсуждения.

Для запуска проекта вам необходимо:
- Копируем репозиторий "git clone https://github.com/Maxmansq/tik-talk.git"
- Устанавлимаем необходимую Node версию "nvm use"
- Устанавливаем Yarn (необязательно) "npm install --global yarn"
- Устанавливаем зависимости "yarn install"
- Запускаем проект "yarn start"


Проект обращается к backend через прокси сервер встроенный в Angular. Файл с прокси находиться в файле "proxy-conf.json".
